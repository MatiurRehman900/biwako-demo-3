## 2. ЯНДЕКС-СПЕЦИФИЧЕСКАЯ SEO-СТРАТЕГИЯ

### Поведенческие факторы (критически важны для Яндекса)
- **CTR:** Заголовки должны содержать цифры (22 года, 2500+ контейнеров), эмоциональные триггеры
- **Время на сайте:** Добавить видео-контент, интерактивные элементы, подробные описания
- **Отказы:** Первый экран должен немедленно отвечать на запрос пользователя
- **Глубина просмотра:** Перелинковка между страницами, блог с полезным контентом

### Региональная привязка (Яндекс.Вебмастер)
```
Настройки → Региональность → Указать: Россия (все регионы)
Приоритетные регионы: Владивосток, Москва, Санкт-Петербург, Екатеринбург, Новосибирск
```

### Яндекс.Вебмастер — чеклист настройки
- [ ] Добавить сайт: https://webmaster.yandex.ru
- [ ] Подтвердить права через HTML-файл или DNS TXT-запись
- [ ] Загрузить sitemap.xml

- [ ] Создать счётчик: https://metrika.yandex.ru
- [ ] Установить код на все 5 страниц (в `<head>`)

  - Цель 1: Клик на "Заказать контейнер" / "Получить предложение"
  - Цель 2: Отправка формы контакта
  - Цель 3: Клик на номер телефона
  - Цель 4: Клик на email
